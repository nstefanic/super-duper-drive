Super Duper Drive

Cloud Storage

  • description Store your files securely.
  • edit_note Write notes and make them available in the cloud.
  • key Save usernames and passwords for websites in a safe place.
  • Super Duper Drive is a Spring Boot MVC application with Thymeleaf templates for front end and H2 database. It uses Spring Security to authenticate users and authorize access to the application.
